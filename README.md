# `shared-clock`

## Install and Run

```
git clone https://github.com/ircam-ismm/shared-clock.git
npm install
npm run dev
```

## Run in production mode (port 80)

```
sudo PORT=80 npm run start
```

## Clients

- normal clients: `http://m.y.i.p/`
- client with controls: `http://m.y.i.p/#controller`
